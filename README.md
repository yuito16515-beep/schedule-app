# スケジュール管理アプリ

## 概要
予定の追加・編集・削除ができるスケジュール管理アプリです。  
localStorage を利用してデータを保存しているため、ページを再読み込みしても予定が保持されます。  
シンプルで使いやすいUIを意識して制作しました。

---

## 使用技術
- HTML
- CSS
- JavaScript
- localStorage

---

## 実装機能
- 予定の追加
- 予定の編集（Update）
- 予定の削除（Delete）
- データの保存（localStorage）
- レスポンシブ対応

---

## 工夫した点
- 配列操作（map / filter）を用いたCRUD処理
- 状態管理のための editId 変数の利用
- 見やすさを意識したシンプルなレイアウト
- スマートフォンでも操作しやすいデザイン
